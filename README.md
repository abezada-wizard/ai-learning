Esta funcion hace una llamada a la API de Anthropic para obtener una respuesta del modelo.

luego analiza si necesita buscar informacion en internet y si es asi, hace una busqueda y devuelve la respuesta.

Luego de eso guarda la respuesta en una lista 

Esto es una prueba de concepto para ver como funciona el tool use y como se puede integrar con una herramienta de búsqueda.
